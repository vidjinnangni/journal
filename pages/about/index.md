Journal est un moteur de blog minimaliste, open source, construit avec Flask et Markdown. Pas de base de données, pas d'interface d'administration : chaque article et chaque page est un simple dossier contenant un fichier `index.md`.

## Pourquoi ce projet

La plupart des CMS imposent une base de données, un back-office et souvent bien plus de complexité qu'un blog personnel n'en a besoin. Journal part de l'hypothèse inverse : des fichiers texte suffisent, et un simple `git push` (ou un `cp`) suffit à publier ou sauvegarder.

## Comment ça marche

Les articles de ce blog en sont eux-mêmes la démonstration : ils expliquent, étape par étape, comment écrire un article, personnaliser l'apparence, changer le titre du site, construire le menu de navigation, ou encore optimiser le référencement et déployer le site en production.

## Code source

Le projet est open source, sous licence MIT. Le dépôt est accessible depuis le lien GitHub en pied de page — n'hésitez pas à l'explorer, le forker, ou proposer une amélioration.

## Contribuer

Les contributions sont les bienvenues : corrections, nouvelles fonctionnalités, améliorations de la documentation. Le fichier `README.md` du dépôt détaille comment démarrer et proposer une pull request.
